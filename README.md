# struktur-data-python
Sistem Informasi Perpustakaan Kampus

Proyek ini adalah program sederhana berbasis Python yang mensimulasikan sistem informasi peminjaman buku di perpustakaan kampus. Program ini dibuat sebagai tugas praktik untuk memahami konsep dasar struktur data Dictionary dan List dalam Python.

📌 Fitur Program

Program ini memiliki fitur-fitur berikut:

1. Menampilkan Koleksi Buku
      Menggunakan Dictionary untuk menyimpan data buku dengan format {Judul: Tahun Terbit}.
2. Simulasi Peminjaman Buku
      Menggunakan List untuk menyimpan buku-buku yang dipilih oleh mahasiswa.
3. Validasi Peminjaman
      Mengecek apakah buku yang dipilih tersedia di koleksi perpustakaan.
4. Ringkasan Transaksi
      Menampilkan daftar buku yang berhasil dipinjam beserta total jumlahnya.
